# HRV Time-Domain Metrics

Script: `examples/hrv/tdmetrics_example.py`

This example loads shared Medicom MTD R-wave timing data, computes RR intervals, removes median-threshold outliers, and reports time-domain HRV metrics with `biosigpy.hrv.tdmetrics`.

Run it from the repository root:

```bash
python examples/hrv/tdmetrics_example.py
```
