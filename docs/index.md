## Python implementation of Biosiglib specifications

Biosigpy is the Python implementation of [Biosiglib](https://github.com/BSICoS/biosiglib) specifications for biomedical signal processing. It belongs to the [BSICoS](https://bsicos.i3a.es/es/) ecosystem together with [Biosigmat](https://github.com/BSICoS/biosigmat), the MATLAB implementation.

!!! warning "Active development"
    Biosigpy is in active development and is not yet distributed as a stable public package. Install it from the source tree when running examples, tests, or local documentation.
